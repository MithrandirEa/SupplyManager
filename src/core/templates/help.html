Centre d'Aide SupplyManager

Introduction

Bienvenue dans la documentation officielle de SupplyManager.

Cette application a été conçue pour simplifier la gestion des stocks de linge, le suivi des fournisseurs et la coordination des commandes. Elle permet une vue d'ensemble rapide de l'état des stocks et facilite les réapprovisionnements.

Première Utilisation

Pour bien démarrer avec l'application, suivez ces étapes initiales dans cet ordre précis afin d'initialiser correctement vos stocks totaux :

Étape 1 : Saisir les commandes en cours (Extérieur)

Avant de compter ce que vous avez dans vos placards, il est crucial de renseigner ce qui est actuellement chez le fournisseur (blanchisserie).

  • Allez dans le menu Fournisseurs ou utilisez l'action rapide "Nouvelle Commande" du Dashboard.
  • Créez des commandes pour tout le linge qui est parti mais pas encore revenu.
  • Pourquoi ? Cela va peupler la colonne "Extérieur" de vos stocks pour que le calcul du total soit juste.
Étape 2 : Saisir l'inventaire physique (Disponible)

Une fois les quantités extérieures définies à l'étape 1, vous devez compter ce qui est physiquement présent.

  • Allez sur le Dashboard, bouton "Mettre à jour l'inventaire".
  • Comptez et saisissez les quantités réelles présentes dans vos locaux (propres et sales, tant que c'est sur site).
  • Le système calculera alors automatiquement votre Quantité Totale de référence (Stock Disponible + Stock Extérieur).
Étape 3 : Créer les comptes utilisateurs

Maintenant que les données sont initialisées, invitez vos collaborateurs.

  • Allez dans le menu "Équipes" via la barre de navigation.
  • Cliquez sur "Ajouter un membre".
  • Créez un compte pour chaque membre du personnel. Attribuez le rôle approprié (Staff pour la saisie simple, Manager pour la gestion complète).

Le Dashboard

Le tableau de bord est votre point d'entrée principal. Il vous offre une vue synthétique des indicateurs clés :

  • Articles en alerte : Liste des articles dont le stock disponible est critique.
  • Commandes en cours : Suivi des commandes envoyées aux fournisseurs mais non encore réceptionnées.
  • Commandes en retard : Alertes sur les commandes dont la date de retour prévue est dépassée.
Astuce : Utilisez les boutons d'actions rapides pour créer une commande ou mettre à jour un inventaire sans changer de page.

Gestion des Stocks (Inventaire)

La gestion des stocks repose sur un principe simple. Chaque article possède :

Type de Quantité Description
Quantité Totale La somme de tout ce que l'établissement possède.
Disponible Ce qui est physiquement présent dans vos locaux, prêt à être utilisé.
Extérieur (Chez le fournisseur) La quantité envoyée en blanchisserie ou en commande.

Le système calcule automatiquement : Total = Disponible + Extérieur.

Lorsque vous effectuez un Inventaire, vous comptez ce qui est physiquement présent. Si le total théorique ne correspond pas à la réalité, le système calculera automatiquement les pertes ou les excédents.

Fournisseurs & Commandes

Cette section permet de gérer vos interactions avec les blanchisseries et fournisseurs de matériel.

Cycle de vie d'une commande :

  1. Création : Sélectionnez un fournisseur et ajoutez des articles. Définissez une date de retour prévue.
  2. En attente (Pending) : La commande est partie. Les articles sont déplacés du stock "Disponible" vers le stock "Extérieur".
  3. Réception : À la réception, indiquez les quantités reçues. Les articles reviennent dans le stock "Disponible". la commande passe en statut "Terminée".

Gestion des Utilisateurs

L'application distingue plusieurs rôles :

  • Staff : Peut consulter les stocks et effectuer des inventaires.
  • Manager : A accès à toutes les fonctionnalités, y compris la création de commandes et la gestion des utilisateurs.
  • Admin : Accès complet à la configuration technique.

Import / Export

Pour faciliter la gestion de grandes quantités d'articles ou la migration de données, vous disposez d'un outil d'import/export compatible Excel et CSV.

Structure du Fichier d'Import

Pour que l'import fonctionne, votre fichier doit respecter les colonnes suivantes (l'ordre n'est pas strict mais les noms doivent correspondre) :

Colonne Obligatoire ? Description
Nom OUI Identifiant unique de l'article (ex: "Drap Housse 140")
Catégorie Non Crée la catégorie si elle n'existe pas (ex: "Linge de lit")
Total Non Quantité totale possédée. Si vide, sera calculé : Disponible + Hors site.
Prioritaire si renseigné.
Disponible Non Quantité stockée sur site. (Défaut : 0)
Hors site Non Quantité chez le fournisseur. (Défaut : 0)
Fournisseurs Non Noms des fournisseurs séparés par des virgules (ex: "Elis, Metro")
Excédent Non Quantité excédentaire relevée.
Informations Non Texte libre.
Astuce : La méthode la plus simple pour obtenir un modèle de fichier valide est de faire d'abord un Export Excel depuis l'onglet Matériels, de le vider, puis de le remplir.
